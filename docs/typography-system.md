# Typography System — ATTAX Solutions

## Font Families

| Role | Family | Usage |
|---|---|---|
| Primary UI | `'Inter Tight', sans-serif` | All body text, labels, buttons, nav |
| Display / Italic accent | `'Playfair Display', serif` | Italic headline words (e.g. "Define You", "Real Clients.") |

---

## Section Heading Sizes (H2)

After audit and fixes applied:

| Section | Mobile | Tablet (`md:`) | Desktop (`lg:`) |
|---|---|---|---|
| Hero (`AttaxHero`) | `32px` | `44px` | `72px` |
| Strategic Platform (`AttaxStrategicPlatformV2`) | `32px` | — | `40px` |
| Services (`AttaxServices`) | `32px` | `36px` | `56px` |
| Results / Testimonials (`AttaxResults`) | `32px` | `42px` | `52px` |
| Clarity (`AttaxClarity`) | `32px` | `34px` | `34px` |
| Mid-CTA (`AttaxMidCTA`) | `32px` | `40px` | `56px` |
| About (`AttaxAbout`) | `32px` | `40px` | `48px` |
| FAQ (`AttaxFAQ`) | `32px` | `42px` | `52px` |
| Footer CTA (`AttaxFooter`) | `32px` | `38px` | `44px` |

> **Pattern:** All H2s start at `32px` mobile. Desktop typically lands between `40px–72px`.
> **Intermediate `md:` step** added during audit to smooth large jumps.

## Label / Eyebrow Text

All section eyebrow labels (e.g. `[Client Results]`, `[From Our Founder]`) use:
- Size: `14px–16px`
- Color: `#1d1ee3` (brand blue)
- Weight: `500`
- Family: `Inter Tight`
