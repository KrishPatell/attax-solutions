# Design System - ATTAX Solutions

## Colors
- **Primary**: `#1d1ee3` (Primary Blue)
- **Primary Dark**: `#1618c7` (Hover states)
- **Background Dark**: `#0a1628` (Hero, Footer)
- **Background Light**: `#f7f9ff` (Alternating sections)
- **Accent Light**: `#a5b4fc` (Italic text, highlights)
- **Border**: `#eaeeff` (Light sections)
- **Glassmorphism**: `backdrop-blur-[20px] bg-white/10 border border-white/20` (CTA banners)

## Typography
- **Headings (Sans)**: Inter Tight (600 for H1, 500-600 for H2/H3)
- **Headings (Serif)**: Playfair Display (600 for italic highlights)
- **Body**: Inter Tight (400 for regular text)
- **Mobile Scale**: 
  - H2: Exactly **32px**.
  - H3: Exactly **26px**.
  - Body: Exactly **15px**.
  - Inner Card Body (PX3s): Exactly **12px**.

## UI Patterns
- **CTA Buttons**: Rounded-full (50px), `pl-6 pr-2 py-2`, icon in white circle.
- **Button Sizing (Mobile)**: Maximum width of **250-280px** on mobile.
- **Section Headers**: 
  - Eyebrow: `[Tag Name]` in `#1d1ee3`, font-medium, 16px.
  - H2: `leading-[1.15] text-[#0a1628] font-medium`.
- **Cards**: `bg-[#f7f9ff]`, rounded-[16px], border-[#eaeeff], p-8.
- **Trust Badges**: Used to establish credibility near CTAs and in the hero.
- **Icon Circles (Contact Lists)**: Exactly **50x50px** rounded-full, `bg-[#EAEEFF]`, text `#1D1EE3`.
- **Spacing**: 
  - Mobile: `py-[60px]`, `px-5` horizontal padding.
  - Desktop: `py-[120px]`, standard grid alignment.
